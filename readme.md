# very_useful_bot
[v1.5.3](#changelog)  
2025.2.1  

如同名稱，這是一個非常有用的Discord機器人，以下是主打的功能：

- ### Wordle
    最好玩的Wordle遊戲現在在Discord上就能玩，而且一天可以玩無限次

- ### C++ Online Judge (線上解題系統)
    一個在Discord上的OJ系統，目前只支援C++

- ### 數學計算
    主要是高中數學，例如約分分數、行列式、向量的內外積等。

- ### 決定吃什麼
    選出下一餐吃什麼。

- #### 單字卡
    複習單字的小工具。

# Very Useful Bot C++ Online Judge System (VUBOJ)
這是一個簡易的線上解題系統 (Online Judge, OJ)，就是輸入程式碼，然後交由系統批改的平台。目前的程式語言僅支援C++，未來會再製作C和Python的版本。


VUBOJ目前不能
- 顯示AC (Accepted) 比例
- 告訴你錯誤訊息，例如NA、WA、TLE等
- 使用除了C++以外的語言
- 發生無限迴圈時可能無法自動停止

運行程式的限制為
- 語言：C++
- 時間：<1s
- 記憶體限制：256MB
